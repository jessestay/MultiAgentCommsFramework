# Live AI Behavior Test Logs and Plans

This directory stores artifacts related to the live testing of AI agent behavior, specifically for verifying adherence to codified `.cursor/rules` that implement complex collaborative workflows.

## Contents:

*   **`MACF_live_test_plan.md`**: (To be created as part of `US-MACF-T01`) The master test plan outlining general methodology, example user prompts for initiating and guiding AI workflows in Cursor, and expected observation points.
*   **`US-ACCEL-01_live_test_log.md`**: (To be created as part of `US-MACF-T01`) Log of a guided live test for the Smart Test Subset workflow (`US-ACCEL-01`).
*   Other `US-*.md` files: Logs for other specific workflow tests.

These documents are critical for ensuring that the AI team not only has well-designed collaborative strategies but can also execute them autonomously and correctly within the Cursor IDE environment based on its `.cursor/rules`.
