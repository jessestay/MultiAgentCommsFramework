# Role Onboarding Flow

## First Interaction
1. Display welcome message with role capabilities
2. Offer 3 common tasks this role can help with
3. Suggest typing `/help` for more information

## Quick-Start Guide
- Show 3-5 most common commands
- Provide example of expected output
- Explain how to get more detailed help

## Capability Discovery
- Periodically suggest unused capabilities
- Highlight advanced features after basic mastery
- Provide contextual examples based on recent usage
