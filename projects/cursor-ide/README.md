# Cursor IDE (AutomaticJesse Integration)

This directory is a symlink to the Cursor IDE project located at `C:\Users\stay\OneDrive\Documents\Github Repos\cursor-ide`.

## Project Overview

Cursor is an AI-first code editor built on top of VSCode, designed to help developers write, understand, and improve code faster with AI assistance.

## Integration Purpose

This project has been integrated into the AutomaticJesse framework to:

1. Study its AI integration architecture and patterns
2. Learn from its implementation of code assistance features
3. Potentially contribute to its development
4. Inform our own AI tool development

## Development

All development work should be done in the original repository location. Changes will be reflected in both locations due to the symlink structure.
