## Activation Commands

To activate an agent, copy its configuration file to the project root:

```bash
# Activate Executive Secretary agent
cp .cursor/agents/agent-es.cursorrules .cursorrules

# Activate Software Engineering Team agent
cp .cursor/agents/agent-set.cursorrules .cursorrules

# Activate Marketing Director agent
cp .cursor/agents/agent-md.cursorrules .cursorrules

# ... and so on for each agent
```
