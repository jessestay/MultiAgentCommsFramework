# MACF Project - Reusable Testing Framework

This directory is the heart of the reusable testing framework for projects utilizing the multi-agent communications system.

## Purpose
The framework aims to provide a standardized set of tools, base classes, common Behat step definitions, helper functions, and conventions to accelerate and simplify the process of writing unit, integration, and acceptance tests across different projects.

## Intended Usage
- Projects should be able to include or reference components from this framework.
- It should provide clear extension points for project-specific testing needs.
- Documentation within this framework will guide developers on how to integrate and utilize its features.

## Contribution Guidelines
- Contributions should be general-purpose and applicable to multiple projects.
- All framework code must be rigorously tested itself.
- Follow TDD/BDD principles for any new framework components.
- Ensure clear documentation for any added utilities or base classes.
