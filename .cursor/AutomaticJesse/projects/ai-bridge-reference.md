## Overview

The AI Bridge provides a secure middleware layer for AI interactions, allowing for:

1. **Flexible Routing**: Automatically routes requests between local and cloud AI services
2. **Content Sanitization**: Provides sanitization of requests and responses
3. **Local Processing**: Option to use Ollama for local AI processing
4. **Cloud Fallback**: Fallback to cloud AI services when needed
5. **Configurable Rules**: User-configurable routing rules and overrides

## Integration with Projects

The AI Bridge can be integrated with any project that needs secure AI interactions. It is currently used by:

- Facebook Growth AI
- Social Media Growth AI
- Future projects that require secure AI interactions
